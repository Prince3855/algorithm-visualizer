# Algorithm Visualizer

### Project Setup
```
1. git clone https://github.com/Prince3855/algorithm-visualizer.git
2. cd algorithm-visualizer
3. npm install
```

### Run Project
```
npm start
```

### Algorithm
#### Sorting Algorithms
1. Bubble sort
2. Selection sort
3. Insertion sort
4. Merge Sort
5. shellsort,heap

#### Path Finding Algorithms
1. Using DFS
2. Using BFS
